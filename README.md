# Calculator Library

This is example of nodejs command pattern with unit testing using mocha and chai

#### Clone Repo

`$ git clone https://github.com/jokosudarsono/nodejs-calculator-example.git`

#### Download Dependecies

`$ npm install`

#### Run Test

`$ npm run test`

#### Run Apps

`$ npm start`

#### Requires

`- node version: ^10.0.0`
